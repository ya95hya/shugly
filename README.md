# شغلي العراق - منصة عمال الأعمال المنزلية

منصة ويب شاملة باللغة العربية تربط بين المستخدمين وعمال الأعمال المنزلية في العراق، مع دعم ثلاثة أنواع مستخدمين (مستخدمون، عمال، مسؤولون).

## المميزات

### للمستخدمين
- البحث عن العمال المتاحين
- فلترة حسب نوع الخدمة والسعر والتقييم
- حجز العمال مع تحديد التاريخ والوقت
- إدارة الحجوزات
- نظام تقييمات ومراجعات
- نظام رسائل مع العمال

### للعمال
- لوحة تحكم شخصية
- إدارة الحجوزات (قبول/رفض/إنهاء)
- تعديل الملف الشخصي والخدمات
- عرض الإحصائيات والأرباح
- التواصل مع المستخدمين

### للمسؤولين
- لوحة تحكم إدارية
- إدارة المستخدمين والعمال
- مراقبة الحجوزات
- إحصائيات عامة

## التقنيات المستخدمة

- **Frontend**: React 18, TypeScript, Chakra UI
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Routing**: React Router
- **Forms**: React Hook Form
- **Notifications**: React Hot Toast
- **Icons**: Lucide React

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
npm install
```

2. إعداد Firebase:
   - إنشاء مشروع جديد في Firebase Console
   - تفعيل Authentication و Firestore
   - نسخ بيانات التكوين إلى `src/firebase/config.ts`

3. تشغيل المشروع:
```bash
npm start
```

## قاعدة البيانات

### المجموعات (Collections)

- **users**: بيانات المستخدمين
- **workers**: بيانات العمال
- **bookings**: الحجوزات
- **reviews**: التقييمات والمراجعات
- **messages**: الرسائل
- **services**: أنواع الخدمات

## الأدوار

- **user**: مستخدم عادي (البحث عن العمال والحجز)
- **worker**: عامل (تقديم الخدمات)
- **admin**: مسؤول (إدارة المنصة)
